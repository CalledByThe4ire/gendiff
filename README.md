## Prerequisites

* Linux, Macos, WSL
* PHP >=8.2
* Xdebug
* Make
* Git

## Addons

Use <http://psysh.org/>

## Setup

```bash
git clone https://github.com/CalledByThe4ire/php-project-48.git
cd php-project-48
make install
```

## Run tests

```sh
make test
```
